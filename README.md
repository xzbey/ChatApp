# Это финальная базовая версия мессенжера с:
* авторизацией
* общим чатом
* личкой (добавление лс только с человеком онлайн)

Всё это работает на локал хосте (захардкожено)
* Сервер авторизации на порту 50000
* Сервер чата на порту 50001
* Сервер валидации данных на порту 50002

Рабочие проги (exe'шники) лежат в `build/Desktop_Qt_6_10_1_MSVC2022_64bit-Release/`:
* сервер чата `/chat_server/chat_server.exe`
* сервер авторизации `/auth_server/auth_server.exe`
* клиент `/client/appclient.exe`

`/Release` не работает

Далее будут доработки дизайна, анимаций, добавление аватарок, токенов и все что с ним связано + авто логина, сейв в бд и мб еще что нибудь, что потом додумаю.
